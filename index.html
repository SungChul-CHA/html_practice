header : 웹 페이지 헤더

nav : 네비게이션 영역

main : 전체에서 가장 핵심되는 내용

div : 블록 요소와 인라인 요소를 그룹으로 묶을 때 사용

blockquote : 문단 단위의 긴 텍스트를 가진 인용문을 작성할 대 사용. 들여쓰기를 해서 출력 출처가 확실한 인용문은 cite 속성으로 출처 경로를 명시

앞 내용 q: 문단 내에 텍스트 단위의 짧은 인용문을 작성할 때 사용. 큰 따옴표로 묶여서 출력 뒤 내용

span : 인라인 요소를 그룹으로 묶을 때 사용
del : 삭제할 텍스트를 표시할 때 사용
ins : 추가할 텍스트를 표시할 때 사용
strong : 텍스트의 의미를 강조하는 태그
em : 텍스트의 강조를 나타내는 태그

용어는 dt 태그
용어 설명은 dd 태그
target 속성 : 링크에 연결된 페이지가 출력될 창. –blank : 새로운 창
action
전송할 서버의 URL 주소
method
전송 방식
get
URL 주소 표시줄에 사용자 입력 값이 노출. 보안 유지가 필요
post
URL 주소 표시줄에 사용자 입력 값이 노출되지 않음. 보안 유지가 필요


legend : 그룹으로 묶은 fieldset 영역에 이름을 붙일 때 사용 fieldset : 관련 있는 폼 요소끼리 그룹을 지을 때 사용

section : 논리적으로 관련있는 콘텐츠 영역. 제목과 함께 사용

select : 콤보박스 option : 선택 항목 추가 optgroup : 그룹으로 묶음

article : 독립적인 콘텐츠 영역.
